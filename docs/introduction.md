# Introduction

PHPTaqwim (a.k.a Taqwim) is a VSCode extension and npm package (CLI) tool that lints and formats PHP code. The word Taqwim stands for "correction" in Arabic.

## ✨ Features

- **Zero Configuration** - Works out of the box with sensible defaults, but fully configurable when needed
- **Lint PHP Files** - Detect syntax errors and coding standard violations
- **Auto-fix Issues** - Automatically fix code according to coding standards
- **Multiple Integrations** - Available as a VSCode extension and an npm package (CLI)
- **No PHP Runtime Required** - No need for PHP to be installed on your system
- **Built-in Presets** - Two presets included: [PSR-12](./rules/psr.md) and [Taqwim](./rules/taqwim.md)

## Why Taqwim?

Taqwim is designed to make PHP development easier by providing a fast and reliable linting and formatting tool that doesn't require PHP to be installed. Whether you're working on a small project or a large codebase, Taqwim can help you maintain consistent code quality and style.

## Getting Started

- For **VSCode users**: Install the extension from the [VSCode Marketplace](https://marketplace.visualstudio.com/items?itemName=kalimahapps.taqwim)
- For **CLI usage**: Install via npm with `npm install @kalimahapps/taqwim -D`

Check out the [Usage](./usage.md) page for more details on how to get started.
